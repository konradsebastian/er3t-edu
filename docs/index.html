About EaR³T

EaR³T (pronounced /ɜːt/) is a modularized Python package developed at the Laboratory for Atmospheric and Space Physics (LASP) and the Department of Atmospheric and Oceanic Sciences at the University of Colorado Boulder. It has been developed over the past decade as part of graduate student education and applied to a wide range of research projects.

EaR³T automates the complete pipeline from satellite data download to 3D radiative transfer simulation. It is inspired by libRadtran and builds on it — using libRadtran as a 1D benchmark solver while extending the framework to 3D solvers (primarily MCARaTS) and real-world satellite data inputs.

📄 Key publication (Chen et al. 2023) 💻 GitHub repository 📚 Documentation

Applications

Radiance Closure

Simulate what a satellite should have measured given retrieved cloud fields, then compare with actual observations. Inconsistency reveals 3D retrieval biases.

OCO-2 · MODIS

Irradiance Closure

Drive surface and aircraft irradiance simulations with geostationary cloud products and compare against direct airborne measurements across entire field campaigns.

CAMP²Ex · AHI · SPN-S

ML Training Data

Generate synthetic 3D-RT radiance fields from LES cloud scenes to train neural networks that learn to correct IPA bias in cloud property retrievals.

CNN · LES · U-Net

Trace Gas Correction

Quantify and correct 3D cloud proximity effects on OCO-2/3 CO₂ retrievals — the first physics-based 3D correction for a trace-gas satellite instrument.

OCO-2 · XCO₂
